# 🛡️ Admin Service

Back-office management dashboard for the NestJS Monorepo.

## Tech Stack
- **React** with Vite
- **@monorepo/api-client**
- **@monorepo/ui** for standard dashboard components

## Features
- **User Management**: Full CRUD for platform users.
- **System Monitoring**: View logs and system health.
- **Dynamic RBAC**: Manage roles and permissions on the fly.

## Development
```bash
npm run dev
```
Accessible at the port defined in Vite config (usually different from web-service).
