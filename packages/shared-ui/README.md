# 🎨 Shared UI

The Design System and component library for the NestJS Monorepo.

## Tech Stack
- **React**
- **Tailwind CSS**
- **Lucide Icons**
- **Framer Motion** for shared animations

## Components
- `Button`
- `Card`
- `Layout` (Shared Shell)
- `StatusBadge` (Admin dashboard specific)

## Development
This package includes a **Storybook** for isolated component development.
```bash
# From the root
npm run storybook --workspace=@monorepo/ui
```
